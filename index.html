About

Was ist die Retrostation?

|

Die Retrostation ist, eine Konsole die es erlaubt einige verschiedene Gamesysteme zu emulieren. Durch die verschiedenen Systeme ist kein Umstecken mehr nötig. Was man allerdings dafür benötigt ist einen Computer damit Mann seine Spielabbilder auf die Konsole bringt. (Achtung: Nur Images von spielen nutzen die man auch im besitzt hat !!!)

|

Die ganzen Konsolen Emulatoren Arbeiten mit der - Retroarchievments - API zusammen d.h ihr könnt euch im Menü mit euren Login Daten anmelden und genauso wie auf den meisten Computer Emus Archievments Sammeln.!

In Zukunft ist auch geplant das 2 Spieler Spiele auch über das Internet Spielbar werden. Das heißt nicht mehr 2 Mann an einer Konsole oder so. Im XMB kommen noch Funktionen hinzu wie Freundesliste, Nachrichten,Archievments Liste u.v.m

|

Kein Ständiges Umgesteckten am TV / oder die ganze Zeit sich mit Emulatoren am Computer Ärgern

FAQs

Warum eine Retroconsole?

Leistungs Stark.

Für kleines Geld gibt es eine echte All in one Console,Die Leistungs Stark genug ist um Spiele Bis zur "PSP" zu Emulieren. Die Konsole ähnelt der Playstation 3 vom Menue her sehr Stark, jedoch kann sie alle gängigen Systeme problemlos wieder geben.!

Verbrauch

Die "kleine" Maschiene kommt mit 5V & 2A vollkommen aus. Mit ihren 4W Strom verbrauch, Verbraucht sie weniger als eine Playstation 1/2/3/4, Xbox,Computer oder sonst irgend etwas.

Die Hardware

Das Board kommt mit einen ArmV7 Cpu mit 4 x 1,2Ghz sehr Flott vorwärts, Der 1GB Ram ist voll kommen ausreichend um GPU sowie das Betriebsystem am Leben zu halten. Zudem hat sie schon Wlan,Bluetooth,Lan,4 Usb Ports an board.

Das System

Ich habe einen Libretro Fork für das Board angefertig, Alles bereits Compeliert und Startbereit, Das einzigste was einen nur noch Fehlt sind die Spiele. Die Spiele bekommt man sehr Leicht im Netz.! Aber Achtung nutzt wirklich nur die Roms die ihr Besitzt.!! alles andere ist Illegal und Zählt meines Wissens nach zu Raubkopien.

Archievment System

Da draußen in der Welt gibt es sehr viel Leute vor dem Computer alte Classiker Spielen, Dank der Retro Archievments Api ist es Möglich sich direkt mit seinen Account Daten auf der Console einzuloggen, Das erspart das Gefummel ständig mit den ganzen Emulatoren.!Die Leute die eine Xbox oder Playstation Besitzen sollten es ja kennen.! Mann muss nur bedenken. Der Account kann nur auf der webseite von Retro Archievments angelegt werden.!

Multimedia Fähig?

Das Xmb kommt mit Media Player, Musik Player, Image Viewer. Somit kann man damit auch mehr als nur Spiele Spielen.! Das "kleine" teil ist in der Lage FullHD Filme Problemlos wieder zu geben. Einen schönen Klaren klang gibt auch der Media Player aus. Noch befindet sich das System im Entwicklungs Modus.Es werden ständig neuerrungen Hinzugefügt und fehler ausgebessert.!
